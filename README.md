# とりあえず読書会

気になった本を勉強会の形式をかりつつ読み進めてみよう —
そんなふうに思い立って YouTube Live にて雑談ライクに読み進めていく読書会です。

## SwiftUI Tutorials

そんな読書会でテーマにした「[SwiftUI Tutorials](https://developer.apple.com/tutorials/swiftui)」を読み進めていく中で仕上がった成果物です。
チュートリアルのコードそのままではなく、Swift Concurrency 対応と、積極的なリファクタリングを行いました。

- Swift API Design Guidelines に基づくリファクタリング
- Swift Macros を用いたリファクタリング（やってみると大袈裟すぎた感）
- Swift Concurrency 対応（ほとんど必要なかった感）
- Swift 6.1 対応
